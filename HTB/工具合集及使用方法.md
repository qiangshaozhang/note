

### gobuster-爆破子域名/目录

GitHub地址：https://github.com/OJ/gobuster

主要有这几种模式

dir - 经典的目录暴力破解模式
dns - DNS 子域暴力破解模式
s3 - 枚举打开的 S3 存储桶并查找存在和存储桶列表
vhost - 虚拟主机暴力破解模式（与 DNS 不同！）

```
gobuster dir -u https://10-10-147-21.p.thmlabs.com/ -w /usr/share/wordlists/dirbuster/directory-list-2.3-small.txt
```
